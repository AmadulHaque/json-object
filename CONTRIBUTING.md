# Contributing

Thank you for considering contributing to the Laravel JSON Object Toolkit!

## Code of Conduct

Please review and abide by our Code of Conduct.

## Contribution Process

1.  **Fork the repository.**
2.  **Create a new branch** for your feature or bug fix.
3.  **Write tests** for your changes. We aim for 100% test coverage.
4.  **Implement your changes.**
5.  **Run tests** using `vendor/bin/phpunit`.
6.  **Submit a Pull Request.**

## Reporting Issues

If you find a bug, please submit an issue with:
- A clear description of the problem.
- A reproduction script or test case.
- Your environment details (PHP version, Laravel version).

## Coding Standards

- Follow PSR-12 coding standards.
- Use strict typing where possible.
- Document public API methods.

## Security Vulnerabilities

If you discover a security vulnerability, please email security@example.com instead of using the issue tracker.
